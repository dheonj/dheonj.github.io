    인코딩: 문자열을 바이트로 변환 #

    • 바이트 코드는 utf-8, euc-kr, ascii 등의 형식을 가짐

    • 파이썬은 문자열을 유니코드로 처리 ㅇㅇ

        즉 파이썬에서 인코딩이란, 유니코드 –> utf-8, euc-kr, ascii #

    디코딩은 그 반대: 바이트를 문자열로 변환 #

        utf-8, euc-kr, ascii –> 유니코드 #
